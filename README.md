# k650di5d3
        本引导为神舟K650di5d3的黑苹果引导文件，
        几乎没有问题了，经过几个月的稳定使用，慢
        慢排错到达现在的完善程度，推荐安装10.11.6，
        其他更高版本也可以，10.12亮度调节档位变化不均匀，可以
        替换10.11.6的亮度驱动稍微好些，但是不能达到10.11.6那样
        的效果，也没有别的问题了。
